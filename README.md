## 1. 产品概述

### 1.1 背景

长时间盯着电脑屏幕会导致眼睛疲劳、干涩、近视加深等问题。调查显示，超过 60% 的上班族每天面对电脑超过 6 小时。

### 1.2 解决方案

遵循国际通用的 **20-20-20 护眼法则**：
- 每工作 **20 分钟**
- 看 **20 英尺（约6米）** 外的远方
- 持续 **20 秒**

### 1.3 产品目标

通过可爱的桌面宠物"眺眺蛙"，用温和、非侵入式的方式提醒用户适时休息眼睛，养成健康的工作习惯。

## 3. 功能规格

### 3.1 核心功能流程

```
                    ┌─────────────────┐
                    │   初始状态      │
                    │  (打招呼界面)   │
                    └────────┬────────┘
                             │ 用户点击"开始护眼"
                             ▼
┌──────────────────────────────────────────────────┐
│                                                  │
│   睡觉状态 (20分钟)                              │
│   - 青蛙在屏幕随机位置漂浮                        │
│   - 右下角时钟显示剩余分钟                        │
│   - 可拖动窗口                                   │
│                                                  │
└──────────────────────────┬───────────────────────┘
                           │ 20分钟倒计时结束
                           ▼
┌──────────────────────────────────────────────────┐
│                                                  │
│   休息状态 (20秒)                                │
│   - 青蛙移动到屏幕中央                            │
│   - 显示"休息一下～"提示                          │
│   - 引导用户看向远方                              │
│                                                  │
└──────────────────────────┬───────────────────────┘
                           │ 20秒倒计时结束
                           ▼
                    ┌─────────────────┐
                    │ 自动进入        │
                    │ 下一个睡眠周期   │
                    └─────────────────┘
```

### 3.2 状态定义

| 状态 | 描述 | 界面元素 |
|------|------|----------|
| GREETING | 初始状态，等待用户点击 | 青蛙图片 + "开始护眼"按钮 |
| SLEEPING | 20分钟护眼中 | 漂浮青蛙 + 时钟 + 暂停按钮 |
| RESTING | 20秒休息中 | 全屏遮罩 + 倒计时 + 青蛙 |
| PAUSED | 已暂停 | 暂停遮罩 + 继续按钮 |

### 3.3 用户操作

| 操作 | 触发方式 | 响应 |
|------|----------|------|
| 开始护眼 | 点击"开始护眼"按钮 | 进入睡眠状态 |
| 暂停 | 点击"暂停"按钮 | 进入暂停状态 |
| 继续 | 点击"继续"按钮 | 恢复睡眠状态 |
| 拖动 | 鼠标拖动窗口 | 移动宠物位置 |
| 退出 | 点击"退出"或托盘菜单 | 关闭应用 |

### 3.4 系统托盘

- **左键单击**: 显示宠物窗口
- **右键单击**: 显示菜单
  - 显示宠物 / 隐藏宠物
  - 当前状态
  - 护眼次数统计
  - 暂停/继续
  - 屏幕遮罩开关
  - Quit 退出

### 3.5 全局快捷键

| 快捷键 | 功能 |
|--------|------|
| ⌘⇧E | 切换宠物窗口显隐 |

## 5. 技术架构

### 5.1 技术栈

| 组件 | 技术 |
|------|------|
| 框架 | Electron |
| 主进程 | Node.js |
| 渲染进程 | HTML/CSS/JavaScript |
| 通信 | IPC (contextBridge) |

### 5.2 文件结构

```
eye-pet/
├── main.js           # 主进程：窗口、托盘、状态机、计时器
├── preload.js        # IPC桥接：安全暴露API给渲染进程
├── index.html        # 渲染进程：UI、CSS、动画
├── package.json      # 项目配置
├── README.md         # 产品文档
└── assets/
    ├── icon.png      # 托盘图标 (22x22)
    └── frog-character.png  # 青蛙形象 (512x512)
```

### 5.3 IPC API

```javascript
// 主进程暴露的API (通过 preload.js)
window.eyePet = {
  getState,           // 获取当前状态
  startCycle,         // 开始护眼周期
  pauseCycle,         // 暂停
  resumeCycle,        // 继续
  toggleOverlay,      // 切换遮罩
  quitApp,            // 退出
  startDrag/drag/endDrag,  // 窗口拖动
  onStateChange,      // 状态变化监听
  onProgressUpdate,   // 进度更新监听
  onOverlayToggle     // 遮罩开关监听
}
```
