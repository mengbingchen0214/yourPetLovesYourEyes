# 护眼Pet / EyePet — 产品规格书 v1.0

## 1. Concept & Vision

**产品名称**：护眼Pet / EyePet

**一句话描述**：用一只桌面宠物提醒你每 20 分钟休息眼睛，治愈又实用。

**核心体验**：
- 一只可爱的宠物长期在你的 Mac 桌面上"生活"
- 安静时轻微浮动，每 20 分钟主动找你互动，提醒你望远护眼
- 支持上传自家宠物照片替换默认形象
- 护眼提醒就像宠物自发的关心，而不是闹钟的机械通知

**用户情绪**：温暖、被治愈、"这只好可爱，我要每天看到它"

---

## 2. 核心功能

### 2.1 护眼提醒系统
- 每 **20 分钟**触发一次休息提醒（遵循 20-20-20 法则）
- 提醒时全屏灰色遮罩覆盖所有显示器，宠物移动到屏幕中央
- 休息持续 **20 秒**，结束后自动进入下一个周期
- 可通过托盘菜单暂停 / 继续

### 2.2 桌面宠物（核心交互）
- 宠物在桌面浮动（无边框透明窗口，始终置顶）
- 可拖动位置
- 双击宠物显示气泡提示
- 睡觉状态有轻微浮动动画，休息状态有点头动画
- 提醒时宠物自动移动到屏幕中央

### 2.3 宠物外观系统
- 默认提供 1 种预设宠物形象（安球）
- 用户可通过托盘菜单分别上传两张图片：
  - **20 分钟睡觉图**：日常陪伴状态
  - **20 秒休息图**：提醒休息时的唤醒状态

### 2.4 宠物图像 Q 版化处理（Web 端）
用户在落地页上传宠物照片，由后端处理后返回 Q 版图片，再应用到桌面宠物：

```
用户上传照片（Web 落地页）
    ↓
FastAPI 后端接收图片
    ↓
rembg 去背景 → 透明底 PNG
    ↓
色彩增强（饱和度 +30%、亮度 +5%、对比度 +10%）
    ↓
返回处理后图片（base64）供下载
```

### 2.5 个性化设置（托盘菜单）
右键托盘图标 → 个性化：
- 更换 20 分钟宠物照（睡觉状态图）
- 更换 20 秒宠物照（休息状态图）
- 更换顶部图标（菜单栏图标）
- 更换 App 图标（Dock 图标）
- 修改问候语（首次启动欢迎文字）

### 2.6 系统托盘
- 左键单击：显示宠物窗口
- 右键单击显示菜单：
  - 显示 / 隐藏宠物
  - 倒计时实时显示
  - 暂停 / 继续
  - 屏幕遮罩开关
  - 个性化子菜单
  - 退出

### 2.7 全局快捷键
| 快捷键 | 功能 |
|---|---|
| ⌘⇧E | 切换宠物窗口显隐 |

## 3. 技术方案

### 3.1 桌面客户端（Mac）
**框架**：Electron + HTML/CSS/JS

**窗口规格**：
- 宠物窗口：200 × 260 px，无边框、透明背景、始终置顶
- 遮罩窗口：覆盖每个显示器全屏，点击穿透，始终置顶

**状态机**：

| 状态 | 描述 |
|---|---|
| GREETING | 初始状态，显示问候语和「开始护眼」按钮 |
| SLEEPING | 20 分钟倒计时中，宠物随机位置浮动 |
| RESTING | 20 秒休息中，全屏遮罩 + 宠物移至中央 |
| PAUSED | 已暂停，保留剩余时间 |

**配置持久化**：用户自定义图片和问候语保存在 `app.getPath('userData')/config.json`

### 3.2 Web 端（落地页 + 图片处理演示）
**框架**：纯 HTML，部署于 Netlify

**功能**：产品落地页 + 宠物图片处理演示（上传 → Q版化 → 下载）

### 3.3 图片处理后端
**技术栈**：Python + FastAPI + rembg + Pillow

**接口**：
- `POST /process-qversion`：去背景 + 色彩增强，返回 base64 PNG
- `POST /remove-background`：仅去背景

### 3.4 提醒系统
- `setTimeout` / `setInterval`（Electron 主进程）
- 暂停时保留剩余毫秒数，继续时从断点恢复

## 4. 商业化路径

**推荐方案**：免费桌面端获取用户 → 小红书内容（宠物+Q版图对比）→ 引流微信私域 → 付费定制 Q 版宠物图（19.9 元/张）

**小红书内容方向**：
- "我的桌面有一只自己的宠物！用自家猫狗生成的 Q 版桌面宠物"
- "每天对着电脑，眼睛还好吗？"
- **关键词**：Mac桌面宠物、自家宠物Q版、护眼提醒、办公室好物
